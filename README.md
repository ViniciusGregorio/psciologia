# psciologia
A repository for a small project about the burnout 

 
 As perguntas foram baseadas nos critérios do Maslach Burnout Inventory (MBI), uma das ferramentas mais utilizadas para diagnosticar o burnout, que avalia três dimensões principais: exaustão emocional, despersonalização e falta de realização pessoal.

Justificativa das Perguntas
"Você sente constantemente que está sobrecarregado no trabalho?"

Mede a exaustão emocional, que é um dos principais sinais de burnout.
"Você tem dificuldade para desconectar do trabalho após o expediente?"

Reflete a dificuldade em separar o trabalho da vida pessoal, um sintoma clássico de burnout.
"Você se sente exausto(a) física e mentalmente frequentemente?"

Avalia o nível de exaustão física e mental, que é central no diagnóstico de burnout.
"Você tem dificuldades para encontrar motivação no trabalho?"

Reflete a desmotivação e a falta de realização pessoal, que são características da despersonalização.
"Você sente que não está recebendo apoio suficiente de colegas ou superiores?"

O apoio social é um fator protetor contra o burnout, e a falta dele é um indicador de risco.
Técnica de Diagnóstico
A técnica usada no teste é baseada em uma adaptação simplificada do Maslach Burnout Inventory e na análise de sintomas psicossociais indicativos de burnout. O cálculo da pontuação é feito somando os pontos de cada pergunta. O nível de risco é classificado como baixo, moderado ou alto, com base na pontuação final.
